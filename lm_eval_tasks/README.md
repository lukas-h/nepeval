# Nepali IndicIFEval lm-eval Tasks

This directory vendors the upstream IndicIFEval `lm-evaluation-harness` custom task files needed for Nepali only:

- `indicifeval_ground_ne`
- `indicifeval_trans_ne`

The task YAMLs intentionally use `test_split: ne` and the copied instruction checker directories contain only `ne_*` files.
